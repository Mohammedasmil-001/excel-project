# Retail Sales Analysis Dashboard (Excel)

## 📊 Project Overview

This project is an **interactive Retail Sales Analysis Dashboard built in Microsoft Excel**.
It helps analyze sales performance across **regions, product categories, sub-categories, and customers** using Pivot Tables, charts, and slicers.

The dashboard provides a quick overview of key business metrics and allows users to filter the data dynamically.

---

## 🚀 Features

* Interactive **Excel Dashboard**
* **Pivot Tables & Pivot Charts**
* **Dynamic Slicers for Filtering**
* Sales analysis by:

  * Region
  * Product Category
  * Sub-Category
  * Customers
* Clean and visually structured dashboard layout

---

## 📈 Dashboard Insights

The dashboard provides the following analytics:

* **Top Customers by Sales**
* **Total Sales by Region**
* **Sales by Sub-Category**
* **Total Quantity of Products Sold**
* **Total Discount by Category**

Users can filter the dashboard using slicers for:

* Product Category
* Region
